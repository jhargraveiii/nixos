# Changelog

- Changed: Added `gpgme` and `gnupg` to `appimage-run` extraPkgs in `global/configuration.nix` — fixes SEGFAULT in `appimageupdatetool` GPG signature validation when updating AppImages (e.g. Cursor) on NixOS (Jim Hargrave, 2026-04-27)
- Fixed: Bluetooth not powered on at boot — added explicit `AutoEnable = true` in `hardware.bluetooth.settings.General` in `global/configuration.nix` to ensure the Bluetooth adapter is enabled on startup (Jim Hargrave, 2026-04-27)
- Changed: Switched `boot.kernelPackages` to `linuxPackages` (default stable) in both `datalore/hardware-configuration.nix` (from `linuxPackages_latest`) and `laptop/hardware-configuration.nix` (from `linuxPackages_6_12`) — reduces rebuild frequency after `nix flake update` since the stable kernel is well-cached on cache.nixos.org and doesn't change as often, avoiding NVIDIA out-of-tree module recompilation (Jim Hargrave, 2026-04-16)
- Fixed: WiFi not detected on boot for datalore — added `iwlwifi` to `kernelModules`, added `iwlwifi power_save=0`, `iwlmvm power_scheme=1`, and `cfg80211 ieee80211_regdom=US` modprobe options to `datalore/hardware-configuration.nix` to ensure Intel WiFi adapter loads reliably at boot (Jim Hargrave, 2026-04-16)
- Changed: Boot log cleanup — simplified `datalore/amd.nix` from 15 amdgpu modprobe options to 2 (kept `runpm=0` and `deep_color=1`), removing 4 invalid params (`use_drm_dp_mst_aux`, `freesync`, `experimental`, `timeout`), 7 redundant defaults, 2 inapplicable GPU-generation flags (`si_support`, `cik_support`), and 3 unnecessary manual DPM udev rules; fixed `/boot` mount security warning by adding `fmask=0077 dmask=0077` in `datalore/hardware-configuration.nix`; removed redundant `bcm5974 debug=0` modprobe option; simplified `datalore/nvidia.nix` by removing `NVreg_EnablePCIeGen3` and `NVreg_UsePageAttributeTable` (handled by modern driver defaults); kept `wpa_supplicant` Wi-Fi backend after `iwd` switch caused deauthentication failures (reason 15: 4-way handshake timeout) with Intel iwlwifi — wext deprecation warning is cosmetic and only affects Wi-Fi 7 hardware (Jim Hargrave, 2026-04-15)
- Removed: Neovim (`modules/programs/neovim.nix`) — removed import from `global/home.nix` and updated direnv `EDITOR`/`VISUAL` from `nvim` to `kate` in `global/configuration.nix` (Jim Hargrave, 2026-04-10)
- Fixed: Build failure caused by removed `nodePackages` set — replaced `nodePackages.vscode-langservers-extracted` with top-level `vscode-langservers-extracted` in `global/home.nix` (Jim Hargrave, 2026-04-10)
- Changed: Silenced Home Manager deprecation warnings in `global/home.nix` — explicitly set `xdg.userDirs.setSessionVariables = true` (keep legacy behavior, ensures XDG dir env vars are available to scripts) and `programs.git.signing.format = null` (adopt new default, no git signing configured) (Jim Hargrave, 2026-03-22)
- Changed: Removed `tsc=reliable` kernel parameter from `laptop/hardware-configuration.nix` — lets the kernel auto-detect TSC reliability on Zen4 and fall back if deep sleep ever desynchronizes it across cores, rather than unconditionally trusting TSC (Jim Hargrave, 2026-03-19)
- Changed: Relaxed TLP `PLATFORM_PROFILE_ON_BAT` from `low-power` to `balanced` in `laptop/configuration.nix` — the BIOS-level `low-power` profile caps total CPU power budget too aggressively, starving WebRTC encoder threads during video calls; `balanced` lifts the firmware power ceiling while still deferring to the `balance_power` EPP hint for per-core frequency decisions (Jim Hargrave, 2026-03-19)
- Changed: Switched `boot.kernelPackages` from `linuxPackages_latest` (6.19.x) to `linuxPackages_6_12` (LTS) in `laptop/hardware-configuration.nix` — kernel 6.12.x is the current LTS with mature amdgpu support for gfx1103, reducing exposure to bleeding-edge driver regressions and scheduler bugs (Jim Hargrave, 2026-03-19)
- Fixed: Root cause of cross-browser video call crashes identified as nixpkgs `strictflexarrays1` hardening flag (PR #473648, Feb 2026) — compiler flag `-fstrict-flex-arrays=1` enabled globally for 5001+ packages, breaking C code using `int arr[1]` flexible array hack in Chromium and Firefox's libwebrtc; added overlay in `overlays/default.nix` to disable `strictflexarrays1` for `chromium` and `firefox-unwrapped` (matches upstream Chromium fix PR #500004 on staging-next, not yet on nixos-unstable); requires source rebuild until fix propagates (Jim Hargrave, 2026-03-19)
- Fixed: Video call crashes (SIGSEGV/SIGTRAP across Firefox, Chromium, Slack) also caused by TLP `CPU_ENERGY_PERF_POLICY_ON_BAT="power"` — the most aggressive EPP hint caused `amd-pstate-epp` to aggressively downclock during WebRTC video encoding, producing invalid bitrate measurements that triggered assertion failures; changed to `balance_power` for sufficient CPU headroom during burst workloads (Jim Hargrave, 2026-03-19)
- Changed: Moved Firefox and Chromium back to unstable — stable pins (Firefox 140.0.1, Chromium 138) had their own bugs; unstable has months of fixes (Jim Hargrave, 2026-03-19)
- Changed: Comprehensive hardware config cleanup — removed no-op TLP settings (`RADEON_*` for legacy radeon not amdgpu, `DISK_APM_*`/`SATA_LINKPWR_*` for SATA not NVMe); removed duplicates across files (`hardware.amdgpu.initrd.enable`, `hardware.graphics.enable`, `clinfo`, `vulkan-tools`, `cppcheck`, `psutils`, `fwupd`, AppImage binfmt registration); removed dead settings (`WLR_NO_HARDWARE_CURSORS` wlroots-only on KDE, `services.hdapsd` ThinkPad-only, `services.blueman` not on KDE, `hardware.enableRedistributableFirmware` implied by `enableAllFirmware`, `boot.supportedFilesystems ext4` default, conflicting `loglevel=4`); removed unused function args from `laptop/configuration.nix` (Jim Hargrave, 2026-03-19)
- Changed: Pinned Chromium and Firefox to `pkgs-stable` — unstable Chromium 146 crashes with SIGILL due to nixpkgs `strictflexarrays1` hardening (nixpkgs#499982), unstable Firefox 148 has GC crashes during video calls; added `pkgs-stable` to `specialArgs` in `flake.nix` for both hosts so NixOS modules can access it; set `programs.firefox.package = pkgs-stable.firefox` in `global/configuration.nix` (Jim Hargrave, 2026-03-14)
- Changed: Removed TLP-redundant power settings from `laptop/hardware-configuration.nix` — removed `vm.laptop_mode`, `vm.dirty_writeback_centisecs`, `vm.dirty_ratio`, `vm.dirty_background_ratio` sysctls (TLP manages disk power via DISK_APM); removed `ahci.mobile_lpm_policy=3` kernel param (TLP manages via SATA_LINKPWR); removed `snd_hda_intel power_save` modprobe (TLP manages via SOUND_POWER_SAVE dynamically per AC/BAT); removed `btusb enable_autosuspend` modprobe (TLP manages via USB_AUTOSUSPEND) (Jim Hargrave, 2026-03-14)
- Fixed: Video conferencing crashes (Firefox SIGSEGV, Chromium/Slack SIGILL) caused by ROCm `HSA_*` env vars (`HSA_OVERRIDE_GFX_VERSION`, `HSA_ENABLE_SDMA`, `HSA_XNACK`) set globally in `laptop/configuration.nix` session variables — moved to `services.ollama.environmentVariables` only; also removed `rocmPackages.clr.icd` and `libva-vdpau-driver` from `hardware.graphics.extraPackages`, removed `hardware.amdgpu.overdrive` and `corectrl` from `laptop/amd.nix`, removed `VDPAU_DRIVER` session variable (Jim Hargrave, 2026-03-14)
- Added: Chromium browser in `global/home.nix` as a Google Meet fallback (Jim Hargrave, 2026-03-14)
- Fixed: MT7921 WiFi boot-time connection failure and random disconnects — changed TLP `PCIE_ASPM_ON_BAT` from `powersave` to `default` to prevent ASPM L1 re-enablement conflicting with `mt7921e disable_aspm=1`; added `mt7921_common disable_clc=1` modprobe option to suppress clock calibration instability; added NM `connection.auth-retries=5` and `wifi.cloned-mac-address=stable` for roaming stability; runtime fix: `nmcli connection modify DATALORE wifi-sec.psk-flags 0` to store PSK in system connection (resolves "no secrets agent" failure before KWallet starts) (Jim Hargrave, 2026-03-14)
- Fixed: Mouse/touchpad lockup on battery — removed `pcie_aspm.policy=powersave` kernel param (was forcing aggressive ASPM at boot before TLP starts, locking I2C HID input devices); TLP already manages ASPM per AC/BAT mode. Removed `options usbcore autosuspend=2` (blanket suspend including input devices); TLP `USB_AUTOSUSPEND=1` handles this with automatic input device exclusion. Removed invalid `mt7921e power_save=0` modprobe param (driver ignores it). Added `sp5100_tco` to blacklisted modules. Cleaned duplicate kernel params (`quiet`, `lsm`) already set by global config or NixOS modules. (Jim Hargrave, 2026-03-11)
- Changed: Relaxed Firefox privacy settings in `global/home.nix` — replaced `privacy.resistFingerprinting` (which spoofs browser identity and breaks "remember this device" for 2FA) with `privacy.fingerprintingProtection` (granular, less disruptive); changed `network.cookie.cookieBehavior` from `1` (block all third-party) to `5` (Total Cookie Protection — isolates per-site instead of blocking, preserving SSO/OAuth flows); added `network.cookie.lifetimePolicy = 0` to ensure session cookies persist (Jim Hargrave, 2026-03-10)
- Fixed: TLP `CPU_SCALING_GOVERNOR_ON_BAT="schedutil"` error — `schedutil` is unavailable with `amd-pstate-epp` driver; changed to `performance`/`powersave` which are the only supported governors for this driver (Jim Hargrave, 2026-03-08)
- Fixed: AMD XDNA (NPU) firmware probe failure after flake.lock update — upstream `linux-firmware` redirected `amdnpu/1502_00/npu.sbin` to an empty v1.5.2.380 placeholder; added `hardware.firmware` overlay in `laptop/configuration.nix` that provides the real v1.5.5.391 blob uncompressed so the kernel finds it first (Jim Hargrave, 2026-03-08)
- Fixed: `pre-shutdown.service` and `pre-sleep.service` boot errors caused by systemd 259 rejecting empty oneshot stubs from upstream `power-management.nix` — added `powerManagement.powerDownCommands = "true"` in `global/configuration.nix` as a no-op workaround (Jim Hargrave, 2026-03-08)

- Added: `nixpkgs-stable` (nixos-24.11) flake input as a fallback package source for broken unstable packages; threaded `pkgs-stable` through `extraSpecialArgs` for both datalore and laptop hosts (Jim Hargrave, 2026-03-05)
- Changed: Switched `gearlever` to `pkgs-stable.gearlever` in `global/home.nix` — upstream `dwarfs-0.12.4` fails to build with boost 1.89 on nixos-unstable (Jim Hargrave, 2026-03-05)

- Fixed: Home Manager activation failure at boot caused by stale `mimeapps.list.backup` blocking backup of `mimeapps.list` — removed stale backup and added `xdg.configFile."mimeapps.list".force = true` in `global/home.nix` to prevent recurrence (Jim Hargrave, 2026-03-05)

- Fixed: MT7921 WiFi random disconnects — added `power_save=0` modprobe option, udev rule to disable PCI runtime PM for the WiFi adapter, and TLP denylist entries for the MT7921 driver and PCI address (Jim Hargrave, 2026-03-04)

- Changed: Removed TPM kernel module blacklist from `laptop/hardware-configuration.nix` after BIOS reset resolved the field failure (Jim Hargrave, 2026-02-18)
- Changed: Added ROCm `gpuTargets` overlay in `laptop/amd.nix` to restrict builds to `gfx1100` only (from 14 default targets), preventing OOM crashes when building hipblaslt/Tensile on the 13GB RAM laptop (Jim Hargrave, 2026-02-18)
- Fixed: Reverted `flake.lock` to previous committed revision to restore binary cache availability for ROCm/ollama-rocm packages (Jim Hargrave, 2026-02-18)
